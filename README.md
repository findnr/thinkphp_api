基于thinkphp6.1开发
使用多应用模式
```sh
composer require topthink/think-multi-app
```
自己封装的库
```sh
composer require findnr/cym-aliyun
composer require findnr/cym-phptools
```
UUID库
```sh
composer require ramsey/uuid
```
图片验证码库
```sh
composer require gregwar/captcha

```
